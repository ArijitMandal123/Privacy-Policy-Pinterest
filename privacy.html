Privacy Policy

Last updated: January 20, 2026

Welcome to PersonalBoardSync (the "Application"). This Application is a personal automation integration developed by Ari Development Projects for the purpose of synchronizing and scheduling content to Pinterest boards.

1. Information We Collect

Our Application is designed for personal automation purposes. We minimize the data we collect to strictly what is necessary for the Application to function.

  • Voluntarily Provided Information: We may collect information that you voluntarily provide, such as image URLs and text descriptions intended for publication.
  • Usage Data: The Application serves as a conduit between your local environment and the Pinterest API. We do not store personal user data.

2. How We Use Your Information

We use the information we collect strictly to authenticate with the Pinterest API on your behalf and to upload Pins to your specified boards.

3. Third-Party Services

We use the Pinterest API to facilitate the posting of content. By using this Application, you acknowledge that your data (specifically images and captions you choose to upload) will be transmitted to Pinterest. Please review Pinterest's Privacy Policy.

4. Data Security

All transmission of information between the Application and the Pinterest API is conducted over HTTPS/TLS encrypted connections. Authentication tokens are stored securely in a local environment.

5. Contact Information

For questions about this policy, please contact the developer via this GitHub repository.
